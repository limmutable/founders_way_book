# Chapter Template

This template provides the standard structure for book chapters.

---

## File: 01-title.md

```markdown
# 01 – Title

## [Chapter Title: Descriptive Name]
```

---

## File: 02-introduction.md

```markdown
# 02 – Introduction

## Introduction

[Brief chapter overview - 2-3 paragraphs explaining what this chapter covers and why it matters]

### Learning Objectives

* [Objective 1 - What readers will learn]
* [Objective 2 - What readers will understand]
* [Objective 3 - What readers will be able to do]
```

---

## File: 03-[topic-name].md (Content Files)

```markdown
# 03 – [Topic Name]

## [Topic Name]

**Definition:** [Brief definition if introducing a new concept]

[Main content - explanations, frameworks, examples]

### Key Points

* [Point 1]
* [Point 2]
* [Point 3]

### Example: [Example Name]

[Real-world example or case study]

**Source:** [Citation with link]
```

---

## Usage Notes

1. **File Numbering:** Use two-digit prefixes (01, 02, ..., 10, 11, etc.)
2. **File Naming:** Use kebab-case: `03-topic-name.md`
3. **Headers:** First line uses format: `# [NN] – [Title]`
4. **Main Heading:** Second line uses format: `## [Section Title]`
5. **Citations:** Always include sources for data, quotes, and claims
6. **Examples:** Use real-world cases with proper attribution
